🚀 Overview

Smart Stock Manager is a modern inventory and analytics platform designed for small businesses, retailers, healthcare suppliers, and nutraceutical companies.

The application provides an intuitive interface for managing products, monitoring inventory levels, tracking sales activities, analyzing revenue trends, and maintaining stock availability.

Built using React and Recharts, the dashboard delivers real-time insights through visual analytics and an interactive user experience.
